# Flask Wireframe Implementation

This is a Flask implementation of the requested wireframe.

## Setup
1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Run the application locally:
   ```bash
   python3 app.py
   ```
3. Run the application in production:
   ```bash
   gunicorn app:app
   ```
4. Open a browser and navigate to `http://localhost:5000` (for local) to view the application.
